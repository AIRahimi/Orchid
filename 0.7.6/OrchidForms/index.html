FormsGenerator

allData

Type: JSONElement

Default Value: {}

Description: All options passed to this object.


baseDir

Type: String

Default Value: "forms"

Description: The base directory to look for forms in.


theme

Type: String

Default Value: empty string

Description: Set a theme to be used only when rendering pages from this Generator.


layout

Type: String

Default Value: empty string

Description: Set the default layout to be used for all Pages from this Generator. Pages can specify their own layouts, which take precedence over the Generator layout.


parallel

Type: boolean

Default Value: false

Description: Improve site generation performance by rendering the pages from this Generator in parallel. (There are currently thread-safety issues that may cause deadlocks, and this should be left false until these issues can be addressed.)


FormComponent

allData

Type: JSONElement

Default Value: {}

Description: All options passed to this object.


extraCss

Type: String[]

Default Value: empty string

Description: An array of CSS Resources to add along with this Component.


extraJs

Type: String[]

Default Value: empty string

Description: An array of JS Resources to add along with this Component.


order

Type: int

Default Value: 0

Description: By default, components are rendered in the order in which they are declared, but the ordering can be changed by setting the order on any individual component. A higher value for order will render that component earlier in the list.


form

Type: Form

Default Value: null

Description: The Form to render. Can be either a key to an indexed form definition, or a complete form definition for a one-off use.


noWrapper

Type: boolean

Default Value: false

Description: When true, this component will not be wrapped in a wrapper element. The wrapper element is determinedby the theme, and it is up to the theme to ensure this is implemented properly.


templates

Type: String[]

Default Value: empty string

Description: Specify a list of templates to use when rendering this component. The first template that exists will be chosen for this component.


hidden

Type: boolean

Default Value: false

Description: When true, this component will not have a template rendered on the page. Useful for Components that only add extra CSS or JS, or for temporarily removing a component from the page.

Form

method

Type: String

Default Value: "POST"

Description: The HTTP request method the form submits as. One of ['GET', 'POST'].


title

Type: String

Default Value: empty string

Description: The user-facing title of the form.


action

Type: String

Default Value: empty string

Description: The 'action' attribute of the form. On forms which have a dedicated submission page, if this is not provided, it defaults to the URL of the submission page.


attributesMap

Type: JSONObject

Default Value: {}

Description: A map of arbitrary attributes to add to the form element.


TextField

type

Type: String

Default Value: "text"

Description: The field type, used to determine the template to render for the field and define additional options for that specific field type.


order

Type: int

Default Value: 0

Description: Form fields are defined in maps which do not have a defined order, and so order in which fields are rendered are not necessarily the order they are declared. Set this property to manually define the ordering of each field.


required

Type: boolean

Default Value: false

Description: Whether this field is required for submission.


label

Type: String

Default Value: empty string

Description: The user-facing label of this field.


placeholder

Type: String

Default Value: empty string

Description: The user-facing placeholder of this field.


spanSm

Type: String

Default Value: "auto"

Description: The number of columns this field should occupy on small screens, out of 12. Can also be 'left', 'right', 'auto', or 'full', which are all 12 columns. The default is 'auto'.


templates

Type: String[]

Default Value: empty string

Description: A list of templates to use for this field, which takes precedence over the normal template. The first template in this lit that exists will be chosen, otherwise falling back to the default if the list is empty or none of the templates are found.


span

Type: String

Default Value: "auto"

Description: The number of columns this field should occupy on large screens, out of 12. Can also be 'left' (6columns, ordered from the left), 'right' (6 columns, ordered from the right), 'auto' (equivalent to 'left'), or 'full' (12 columns) The default is 'auto'.
