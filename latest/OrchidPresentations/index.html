PresentationsGenerator

allData

Type: JSONElement

Default Value: {}

Description: All options passed to this object.


baseDir

Type: String

Default Value: "presentations"

Description: The base directory to look for presentation slides in.


theme

Type: String

Default Value: empty string

Description: Set a theme to be used only when rendering pages from this Generator.


layout

Type: String

Default Value: empty string

Description: Set the default layout to be used for all Pages from this Generator. Pages can specify their own layouts, which take precedence over the Generator layout.


parallel

Type: boolean

Default Value: false

Description: Improve site generation performance by rendering the pages from this Generator in parallel. (There are currently thread-safety issues that may cause deadlocks, and this should be left false until these issues can be addressed.)


PresentationComponent

allData

Type: JSONElement

Default Value: {}

Description: All options passed to this object.


deckTheme

Type: String

Default Value: "web-2_0"

Description: The Deck.js presentation theme to use. Should be one of ['neon', 'swiss', 'web-2_0']


extraCss

Type: String[]

Default Value: empty string

Description: An array of CSS Resources to add along with this Component.


extraJs

Type: String[]

Default Value: empty string

Description: An array of JS Resources to add along with this Component.


order

Type: int

Default Value: 0

Description: By default, components are rendered in the order in which they are declared, but the ordering can be changed by setting the order on any individual component. A higher value for order will render that component earlier in the list.


noWrapper

Type: boolean

Default Value: false

Description: When true, this component will not be wrapped in a wrapper element. The wrapper element is determinedby the theme, and it is up to the theme to ensure this is implemented properly.


presentationKey

Type: String

Default Value: empty string

Description: The key of the Presentation to display.


transitionTheme

Type: String

Default Value: "horizontal-slide"

Description: The Deck.js transition theme to use. Should be one of ['fade', 'horizontal-slide', 'vertical-slide']


templates

Type: String[]

Default Value: empty string

Description: Specify a list of templates to use when rendering this component. The first template that exists will be chosen for this component.


hidden

Type: boolean

Default Value: false

Description: When true, this component will not have a template rendered on the page. Useful for Components that only add extra CSS or JS, or for temporarily removing a component from the page.
